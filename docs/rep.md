# <rep

Shows your rep or someone elses.

## Usage

(`variable`) = Required variable.

[`variable`] = Optional variable.

```
<rep [user]
```

Examples:

- `<rep` - Returns your rep.

- `<rep @Discord#0000` - Returns a different users rep. 

## Permissions

Who can use this command:

- Everyone

Who can't use this command:

- *None*
